# Hedera — God Transfer Tracker

Monitors Hedera mainnet for **single transactions moving 500 million+ HBAR** (~$50M–$200M+).

On Hedera, normal people don’t move this much.  
Only three entities can:

- Governing Council
- Treasury (0.0.2)
- Swirlds / Leemon himself
